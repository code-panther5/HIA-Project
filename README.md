HIA – Health Insights Agent

AI powered system that analyzes medical lab reports and generates simplified health insights.

## Features
- OCR based PDF report reading
- Lab value extraction
- AI analysis using LLM
- User authentication (Supabase)
- Health summary generation
- Streamlit interface

## Tech Stack
Python, Streamlit, Supabase, OCR, LLM APIs

## How to Run
pip install -r requirements.txt
streamlit run src/main.py
